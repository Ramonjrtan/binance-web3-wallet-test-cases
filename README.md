# 🧪 Binance Web3 Wallet Test Cases (QA Portfolio)

## 📌 Overview

This repository contains structured manual test cases for **Web3 Wallet features**, designed as a QA portfolio project.

Unlike basic exchange testing, this focuses on:

* Wallet logic
* Security and recovery
* Transaction validation
* dApp interactions

---

## 🔄 Scope

Covered modules:

* Wallet Setup
* Backup & Recovery
* Security & Access
* Assets & Networks
* Send & Receive
* dApp Connection & Signing
* End-to-End Scenarios
